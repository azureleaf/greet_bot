LINE Bot「仙台おでかけアシスタント」の動作を説明します！
GitHub

この動画内では次の順序で操作をしています。わかりやすくするため動画はスロー再生です。

  • GPS位置を送ります。
  • 「地下鉄」をボタンで選択。
  • 地下鉄の路線をボタンで選択。
  • 「現在位置から一番近い、その路線の駅」
    「現在位置から駅までの直線距離」
    「もうすぐ来る直近３つの発車時刻（平日＆休日）」
    が表示されます。
  • GPS位置を送ります。
  • 「バス」をボタンで選択します。
  • 「現在位置から近いバス停の名前＆そこまでの距離」が表示されます。
  • テキストメッセージを送ります。（中身は何でも可）
  • 「現在の天気」
    「３時間毎の予報（１日分）」
    が表示されます。
